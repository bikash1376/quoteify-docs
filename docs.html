Quoteify API - Documentation

Base URL

The base URL for accessing the Quoteify API is:

https://quoteifyapi.onrender.com/quotes/


Endpoints

  • GET /quotes/random: Retrieve a random quote in JSON format.
  • GET /quotes: Retrieve all quotes available in the API in JSON format.
  • GET /quotes/:id: Get a specific quote by its ID in JSON format.


API Usage

To use the Quoteify API, send HTTP requests to the appropriate endpoints mentioned below.


Example Usage - Get Random Quote

Request:

GET https://quoteifyapi.onrender.com/quotes/random

Response:

{
"id": 42,
"text": "Let me live, love, and say it well in good sentences.",
"author": "Sylvia Plath"
}
